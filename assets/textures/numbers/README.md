# assets/textures/numbers — brojevi za dresove

`numbers_*.png` — brojevi koji idu na dresove figurica (još se ne koriste u kodu).
Kasnije ih mapiramo na dres materijal ili kao decal/label iznad igrača
(trenutno `player_appearance.gd` crta broj kao `Label3D`; ovi PNG-ovi su alternativa
za pravi broj na dresu).
